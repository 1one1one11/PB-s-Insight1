# PB's Pick

제가 관심 있게 보는 섹터, 종목, 상품 아이디어를 정리하는 문서입니다.

## 1) 선별 기준

- 설명 가능한 비즈니스인지
- 현금흐름과 재무 체력이 버티는지
- 내러티브보다 실적 검증이 가능한지
- 밸류에이션과 기대치가 과열되지 않았는지

## 2) 체크 프레임

| 항목 | 질문 |
| --- | --- |
| 산업 | 지금 산업의 이익 사이클은 어디쯤인가 |
| 기업 | 경쟁우위가 유지되는가 |
| 재무 | 부채, 현금흐름, 마진이 안정적인가 |
| 촉매 | 향후 6~12개월 이벤트가 있는가 |
| 리스크 | 가설이 깨지는 신호는 무엇인가 |

## 3) 실행 방식

- 한 번에 전부 사지 않고 분할 접근
- 처음에는 `작게 시작하고 근거가 확인되면 키운다`
- 손절보다 `가설 훼손 여부`를 먼저 점검

## 4) 기록 습관

1. 왜 보는지 한 줄로 적기
2. 반대 논리 먼저 적기
3. 다음 점검 시점 정하기
4. 매수/비매수 모두 기록 남기기
