# PB Insight

PB와 고객의 자산 목적에 맞는 전략을 함께 설계하고 실행하는 문서 허브입니다.

이 공간은 PB 실무 운영 방식, 고객 커뮤니케이션 아이디어, 그리고 투자 관점을 구조적으로 정리하기 위해 만들어졌습니다.

## 안내 페이지

### ONLY FOR PB

- 문서: [Only for PB](ai-presentation/only-for-pb.md)
- 설명: 내일 바로 실행할 행동 목표, 우선 고객, 시간 루틴, 핵심 원칙을 한 화면에 모아 둔 집중 보드입니다.

### BEST PB MIND

- 문서: [BEST PB MIND](ai-presentation/best-pb-mind.md)
- 설명: 상위 PB가 가져야 할 태도와 고객 인식 구조를 정리한 기준 문서입니다.

### PB Tier System

- 문서: [PB Tier System](ai-presentation/pb-tier-system.md)
- 설명: 고객 티어 기준, 서비스 차등, 응대 우선순위를 정리합니다.

### PB's Pick

- 문서: [PB's Pick](ai-presentation/pbs-pick.md)
- 설명: 섹터, 종목, 상품 아이디어와 투자 판단 기준을 정리합니다.

### PB's Idea

- 문서: [PB's Idea](ai-presentation/pbs-idea.md)
- 설명: 리서치 기반 운용 아이디어와 상담 연결 포인트를 공유합니다.

### PB Insight Style

- 문서: [PB Insight Style](ai-presentation/pb-insight-style.md)
- 설명: 문서 구조와 메시지 톤을 어떤 방식으로 가져갈지 정리합니다.

### Weekly Report

- 문서: [Weekly Report](ai-presentation/pb-weekly-report.md)
- 설명: 매주 반복 점검할 실적, 고객, 파이프라인 체크 항목을 정리합니다.

## 운영 원칙

- 많이 아는 것보다 많이 실행하는 것이 우선이다.
- 모든 고객보다 자금이 움직일 고객에 집중한다.
- 상담 없는 영업보다 상담으로 이어지는 영업을 만든다.
- 기록이 기억보다 강한 경쟁력이다.
- 단기 실적과 장기 신뢰 사이의 균형을 지킨다.
